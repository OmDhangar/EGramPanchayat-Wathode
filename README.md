## Citizen Services

- Apply and download certificates:
  - Birth Certificate
  - Death Certificate
  - Marriage Certificate
  - *(Upcoming: Caste, Income, Residence Certificates)*
- House Tax System:
  - View pending tax
  - Online payment *(upcoming)*
  - Download receipts *(upcoming)*
- Track application status
- Receive notifications and announcements
- User-friendly Marathi & English interface

---

## Admin Features

- Approve/reject applications digitally
- Manage citizen data and certificate records
- Handle house tax and payment details
- Publish local notices and government schemes
